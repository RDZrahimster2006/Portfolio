Missions réalisées

Ce stage m'a permis de pratiquer Angular et TypeScript dans un contexte professionnel réel, et de comprendre comment une équipe organise et priorise ses développements. J'ai également appris à travailler sur un code existant, sans repartir de zéro.

Stage 2 — H.Bell Consulting · Janvier – Mars 2026 · 8 semaines

Stage approfondi au sein de la même entreprise, H.Bell Consulting, avec cette fois une mission de développement de nouvelles fonctionnalités et d'optimisation d'une application existante — le Habit Tracker. Ce stage m'a permis de pratiquer l'architecture MVC Laravel sur un projet réel avec de vrais utilisateurs.

Projet — Habit Tracker

Application web permettant aux utilisateurs de créer et suivre leurs habitudes quotidiennes, hebdomadaires ou mensuelles. Elle affiche la progression sur les 30 derniers jours et calcule automatiquement les taux de réussite. J'ai repris l'application existante pour y ajouter des fonctionnalités et optimiser le code PHP Laravel ainsi que la base de données.

Missions réalisées

  • Stage approfondi au sein de l'entreprise — montée en compétences rapide
  • Développement de nouvelles fonctionnalités sur le Habit Tracker
  • Optimisation du code PHP Laravel existant (performances, lisibilité)
  • Amélioration et évolution de la base de données MySQL
  • Déploiement de l'application en environnement cloud

Architecture & Base de données

  • users — gestion des comptes utilisateurs
  • taches — stockage des habitudes créées
  • users_taches — association utilisateur / tâches
  • realisations — enregistrement des habitudes complétées

Architecture MVC Laravel : Modèles → Contrôleurs → Vues Blade

Technologies utilisées

Laravel PHP Blade Bootstrap JavaScript MySQL HTML / CSS Git
